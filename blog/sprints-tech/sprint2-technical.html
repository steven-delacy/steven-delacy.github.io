Classes & ID's, but when should you use them?

What is a class?

A class is an element which can be applied to any aspect of HTML. This could mean one to two lines of code, or 50. The class tag once used in css / javascript will apply the same rules to anything that has that rule set applied.

What is an ID?

An ID is similar to a class but can only be used on one selected tag. Again, it can be used in all aspects of HTML and then manipulated with .css or .js.

Example: Say you wanted to style that same paragraph/ title but you didn’t want it to effect any of the others on different pages of what you are working on. Well, we’d use an ID to define it, basically singling it out. Then we can style it with .CSS/ . JS however using those that have been targetted with these elements.
